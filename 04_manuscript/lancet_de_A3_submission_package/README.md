**Branch.** `lancet-A3-methodological` · **Phase.** Day 15 init (per meta-prompt §10·2) · **Date.** 2026-05-09

This is the **procedural / translational layer (L3)** of the FDEP-TP trilogy for *The Lancet Diabetes & Endocrinology*. Companion articles:
- **A1** conceptual / physiological layer — branch `lancet-A1-conceptual`
