## 3 · Abstract

**Paste the 300-word abstract from `EPA_FDEP_TP_manuscript_v3.qmd` lines 14–22 verbatim:**

```
Background. Data-driven taxonomies of metabolic disease classify individuals from
static cross-sectional variables and discard the temporal geometry of the periprandial
response. The Functional Dynamic Enteropancreatic Phenotyping during the Periprandial
Transition (FDEP-TP) framework recasts the joint enteropancreatic response as a
multivariate functional object decomposable into four eigenfunctions (Ψ1 distal
L-cell, Ψ2 proximal-distal sequencing, Ψ3 biphasic coupling, Ψ4 ghrelin tone). We
report its first cohort-level empirical validation.

Methods. Pre-registered ecological analysis (OSF 10.17605/OSF.IO/3CZRE) of 23 source
studies × 1840 subjects across nine canonical strata and 11 analyte forms. Pseudo-
individual-participant data were generated by Gaussian-process AR(1) kernels (ρ=0·5).
The pipeline applied PACE univariate FPCA per hormone, cross-covariance kernel
estimation C^(jk)(s,t), Happ-Greven multivariate FPCA on a broad-coverage 4-hormone
joint operator (TOTAL GIP, ACTIVE GLP-1, TOTAL GLP-1, Glucagon) with Chiou
normalisation, cross-cohort descriptors ρ_INC/ρ_NET/ρ_ANR, and PTP nine-label / IEP
eight-rule classification. Inference: Pillai trace, Mahalanobis distances on
(ξ1..ξ4), bootstrap stability, leave-one-study-out sensitivity.

Findings. The joint block contained 350 pseudo-subjects. Happ-Greven mFPCA recovered
the four canonical eigenfunctions with cumulative FVE 98·5 %. Cohort scores
reproduced the predicted separations: POST-RYGBP a 33-unit ξ1 shift, POST-SG a
distinctive ξ4 elevation (+4·27 vs −1·60 in LEAN), OB+T2DM maximal ξ3 disruption.
ρ_INC dropped from 0·63 (LEAN) to 0·12 (OB) and rebounded to 0·75 post-SG; ρ_NET
reached 0·78 in POST-RYGBP. Modal IEP labels matched predictions: LEAN Type I·I,
POST-RYGBP / POST-SG / OB+T2DM / T2DM Type V·II·b, POST-CR Type II·II. Pillai
pairwise q vs LEAN was <1e-29 in every contrasted stratum.

Interpretation. The analysis validates the FDEP-TP framework across the four
canonical signatures. The bariatric phenotype is bariatric-dominant rather than
RYGBP-exclusive — both POST-SG and POST-RYGBP reach Type V — distinguishing it
from disease and reference cohorts on a continuous Type axis. The framework
operationalises agonist selection and metabolic-surgery indication; thresholds are
hypothesis-generating and require prospective TRIPOD+AI individual-participant
validation.

Funding. None.
```
